# HACK Club KUET – Club Management System

A complete web platform for managing club activities: event registration, team coordination, announcements, and member communication. Built with PHP (PDO), MySQL, and a responsive frontend.

---

## ✨ Features

### Member Dashboard
- Secure login / registration (session‑based)
- Browse upcoming events with seat availability and team participation
- Register for events (with flash success/error messages)
- View your registered events and join status
- Read club announcements
- Send messages to admin (stored in database)
- View your profile (student ID, department, role, join date)
- “Remember me” cookie and theme cookie

### Admin Panel
- Overview statistics (members, events, registrations, messages, teams)
- Manage members: change roles (admin/member), delete users
- Manage events: create, edit, delete, assign seat limits, add/remove participating teams
- Manage announcements: post, edit, delete
- Manage teams: create, edit, delete, add/remove members, assign teams to events
- View and delete member messages (inbox)
- View and delete contact form messages
